##Fibonacci

By Kayla Morrison

### Description
calculates the fibonacci of a number n

### Usage

Replace n with the index for which you would like to find the fibonacci number

``puts fibonacci(n)``

example:

`puts fibonacci(12)` => 144

`puts fibonacci(0)` => 0
